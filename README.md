# 🔐 JavaScript Security Agent

An autonomous AI agent that performs hybrid static + dynamic analysis 
to detect security vulnerabilities in JavaScript codebases.

## 🚀 What It Does
- Detects vulnerabilities using a hybrid static + dynamic analysis pipeline
- Uses RAG to provide contextual explanations and fix suggestions per issue
- Agentic LangChain workflow with multi-step reasoning and tool orchestration
- Structured output: vulnerability type, severity level, and fix recommendation

## 💡 How It Works
1. Input: JavaScript codebase or single file
2. Agent runs static analysis (AST pattern matching) + dynamic checks
3. RAG retrieves relevant CVE and vulnerability context
4. Structured output with vulnerability type, severity, and fix suggestion
